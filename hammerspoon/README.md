# Hammerspoon

## Window Management

**M5** is the bottom-left custom key on the Keychron Q11 left half (sends F18).

| Keys | Action |
|------|--------|
| M5 + ← / → / ↑ / ↓ | Nudge window 100px |
| M5 + Shift + ← / → / ↑ / ↓ | Snap to screen edge |
| M5 + ] | Grow wider (right edge →) |
| M5 + [ | Shrink narrower (right edge ←) |
| M5 + = | Grow taller (bottom edge ↓) |
| M5 + - | Shrink shorter (bottom edge ↑) |

Nudge and resize repeat while the key is held.
