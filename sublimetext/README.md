# Sublime Text

Managed Sublime Text settings, key bindings, snippets, and compatibility wrappers.

What this directory does:

- links repo-managed files into `~/Library/Application Support/Sublime Text/Packages/User`
- provides local `subl` and `mate` wrapper commands through `sublimetext/bin`
- keeps TextMate-era snippets and selected commands available inside Sublime Text

Run `just install` after changing this directory.
